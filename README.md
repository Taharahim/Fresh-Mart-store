# FreshMart - Grocery E-Commerce HTML Template

FreshMart is a modern, responsive grocery e-commerce website template built with HTML, CSS, and Vanilla JavaScript.

It is designed for developers, freelancers, and beginners who want a ready-to-use UI for grocery or food-based e-commerce projects.

---

## 🚀 Features

- Multi-page layout (Home, Products, Cart, Checkout, Contact)
- Product listing system
- Category filtering
- Add to cart (localStorage based)
- Responsive design (mobile + desktop)
- Clean UI structure
- Easy customization

---

## 📁 Included Files

- HTML Pages
- CSS Stylesheets
- JavaScript Logic
- Product Data File
- Documentation
