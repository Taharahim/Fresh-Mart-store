# Installation Guide

## Step 1: Download & Extract
Download the ZIP file and extract it on your system.

## Step 2: Open Project
Open the extracted folder using any code editor (VS Code recommended).

## Step 3: Run Project
Simply open `index.html` in your browser.

No build tools, no dependencies required.

## Step 4: Optional Hosting
To host online:

- Upload all files to your hosting provider
- Or use GitHub Pages
- Or Netlify drag & drop method

## Notes
- Do not remove folder structure
- Keep assets folder intact for images, CSS, and JS
