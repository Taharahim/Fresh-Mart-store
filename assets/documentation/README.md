# FreshMart Grocery Template

## Overview
A clean, modern grocery e-commerce HTML template.

Built with:
- HTML
- CSS
- Vanilla JavaScript

No frameworks. No backend. Just frontend clarity.

---

## Features

- Product grid system
- Product detail support
- Cart with localStorage
- Category filtering
- Search functionality
- Responsive layout
- Clean SaaS-style UI

---

## Folder Structure

- `/css` → styling
- `/js` → logic
- `/data` → product dataset
- HTML files → pages

---

## Run Locally

Just open `index.html` in browser.

Or use Live Server in VS Code.

---

## Deploy

### Netlify
1. Drag folder into Netlify dashboard
2. Done

### Vercel
1. Import project
2. Deploy static site

---

## Editing

- Change products in `/data/products.js`
- Adjust colors in `/css/style.css`
- Modify layout in HTML files
