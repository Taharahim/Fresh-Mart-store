# Fresh Mart

Modern Grocery E-Commerce Website Template

## Overview

Fresh Mart is a responsive grocery e-commerce template built with HTML, CSS, and Vanilla JavaScript.

The template includes:

* Responsive design
* Product listing
* Category filtering
* Search functionality
* Shopping cart
* Local storage support
* Clean modern UI
* Mobile-friendly layout

## Features

* Homepage
* Products page
* Categories
* Shopping cart
* Product filtering
* Search products
* Responsive navigation
* Modern design system

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

## Installation

1. Download template files
2. Extract ZIP archive
3. Open project folder
4. Open index.html in browser

No build tools required.

## 3. Customization Guide

### Change Website Name

Open HTML files and replace:

FreshMart

with your desired brand name.

### Change Logo

Replace logo image inside:

assets/images/

### Change Hero Section

Edit hero section content inside:

index.html

---

## 4. Product Editing Guide

Open:

assets/js/products.js

Example:

{
id:1,
name:"Fresh Apples",
price:4.99,
category:"Fruits",
image:"assets/images/products/apple.jpg"
}

Edit values as needed.

Add more products by duplicating product objects.

---

## 5. Color Editing Guide

Open:

assets/css/style.css

Root Variables:

:root {
--primary:#22c55e;
--secondary:#15803d;
--dark:#111827;
--light:#ffffff;
}

Modify colors and save.

Changes automatically apply site-wide.
